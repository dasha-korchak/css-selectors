Типи селекторів

Тег селектор

На приклад, h1 - задасть властивості css всім елементам <h1> на сторінці

#id

#elem - задасть властивості css всім елементам з id="elem" на сторінці

.class

.selector - задасть властивості css всім елементам class="selector" на сторінці

.class1.class2

.card1.card2 - задасть властивості css всім елементам class="card1 card2" на сторінці

.class1 .class2

.card1 .card2 - задасть властивості css всім елементам class="card2", які вкладені в елемент з "card1"

element, element

div, p - задасть властивості css всім div та всім p на сторінці

element element

div p - задасть властивості css всім елементам <p>, що знаходяться всередині <div>

element>element

div>p - задасть властивості css елементам <p>, що є прямими нащадками <div>

element+element

div+p - задасть властивості css елементам <p>, що слідкують одразу за <div>

element~element

div~p - задасть властивості css всім елементам <p>, що знаходяться після <div>
